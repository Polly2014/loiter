## 技术亮点

### 1. 从本地 Mac 到云端一键搬迁

原设计：Mac 当 AP + broker + server（活动现场离线包）。Polly 一句"既然有了联网地址可以访问，为什么还需要本地 Mac"→ 当场搬到 Azure VM。

**结果**：Cardputer 连任何 WiFi（家/公司/4G）都能进房间，大屏任何人任何时间打开 loiter.polly.wang 都能看。Mac 关机不影响。

### 2. AI 头像双版本 + 三次风格迭代

```
/face cool cat → Azure gpt-image-1.5 (17s)
                 → 大屏 256px 彩色 PNG (WS push)
                 → Cardputer 16×16 1-bit bitmap (MQTT QoS1)
```

prompt 在一晚上迭代了 4 次：
- 黑白像素 → chibi anime → RPG sprite → **水墨画风**

### 3. 稀有度系统（零后端改动）

前端直接按 `msg_count` 算边框：铜(0) → 银(5) → 金(15) → SSR 彩虹(30)。
金框发光、SSR 彩虹流光（hue 旋转）。完全前端渲染，服务端不知道"稀有度"这个概念。

### 4. WS keepalive 解决 Cloudflare tunnel idle cut

大屏 WS 只读、永不发消息 → 空闲 30s 被 tunnel 切 → 不停闪空。
解法：服务端 `asyncio.wait_for(receive, 20s)` → 超时发 `{type:"ping"}`。前端忽略未知 type。

### 5. MQTT 密码 fail-closed

```python
# config.py — 空字符串 = 不鉴权（仅本地开发）
MQTT_USER = os.getenv("LOITER_MQTT_USER", "").strip()
MQTT_PASS = os.getenv("LOITER_MQTT_PASS", "").strip()
```

凭据存 `/etc/loiter/loiter.env` (chmod 600) + firmware `config.h` (gitignored)。
公网 `mosquitto_pub` 无密码 → `not authorised`。

### 踩坑记录

#### 13. ES8311 Speaker.tone() 在 Cardputer-Adv 上不可用
```
Speaker.tone(freq, duration) → 内部启动 FreeRTOS I2S task
↓ 在 MQTT 回调 onMqttMessage() 里调用
↓ I2S task 与 PubSubClient 网络线程争抢 CPU
↓ 看门狗超时 / MQTT keepalive 过期 → 卡死或掉线
```
**教训**：ESP32 I2S 音频不能在 MQTT 回调上下文触发。需要用主循环定时器驱动。

#### 14. MQTT "already connected" 循环踢连
```
WiFi 微抖 → PubSubClient.connected() 返回 false
→ mqttEnsure() 每 3s 尝试 mqtt.connect()
→ broker 踢掉旧连接："already connected, closing old"
→ 旧 LWT 触发 → 新连接又被干扰 → 循环
```
**修复**：`mqtt.disconnect()` before reconnect + keepalive 30→60s + interval 3→5s

#### 15. anon uid="anon" 被 auto-join 创建幽灵成员
服务端 `_handle_anon` 发 `uid=anon` 到 `msg/main` → broker 回传 → `_handle_msg` auto-join → 大屏出现 "???" 角色。
**修复**：`if uid == "anon": return`（和 NPC 同一处跳过）

#### 16. 手机热点 5GHz 不被 ESP32-S3 扫到
ESP32-S3 只支持 2.4GHz WiFi。iPhone "最大化兼容性" / Android "2.4GHz" 切换后解决。
